# Playwright Quick Start

## First Time Setup

```bash
# Install Playwright browsers (required once per machine)
npx playwright install
```

## Running Tests

```bash
# Interactive UI mode (recommended)
npm run test:e2e:ui

# Run all tests (headless)
npm run test:e2e

# Watch tests execute (headed mode)
npm run test:e2e:headed

# Debug mode with inspector
npm run test:e2e:debug
```

## Specific Browser Tests

```bash
npm run test:e2e:chromium
npm run test:e2e:firefox
npm run test:e2e:webkit
```

## View Results

```bash
# Show HTML test report
npm run test:e2e:report
```

## Test Specific Files

```bash
# Run only auth tests
npx playwright test tests/auth.spec.ts

# Run only episode tests
npx playwright test tests/episodes.spec.ts

# Run tests matching pattern
npx playwright test -g "should allow user signup"
```
